# Operaciones Bitwise en Arduino

## Descripción

Proyecto enfocado en la implementación de operaciones a nivel de bits en Arduino, permitiendo manipular datos binarios de forma eficiente.

---

## Características

* Operaciones AND, OR, XOR
* Desplazamientos de bits
* Manipulación de registros

---

## Componentes

* Arduino
* LEDs (opcional para visualización)

---

## Funcionamiento

Se aplican operaciones bitwise sobre variables para demostrar cómo se manipulan los datos a nivel binario dentro del microcontrolador.
