
Sistema Inteligente de Monitoreo de Temperatura

Este proyecto consiste en el desarrollo de un sistema embebido de monitoreo de temperatura utilizando Arduino Uno, sensores analógicos TMP36 y LM35, una pantalla LCD 16x2 y LEDs indicadores. El sistema permite medir la temperatura ambiente en tiempo real, procesar los datos mediante conversión analógico-digital (ADC) y visualizar la información en la pantalla LCD.

Además, el sistema clasifica la temperatura en diferentes rangos térmicos utilizando indicadores visuales:

LED azul para temperaturas bajas (<20°C).
LED verde para temperaturas normales (20°C – 30°C).
LED rojo para temperaturas altas (>30°C).

La simulación virtual fue desarrollada en Tinkercad utilizando el sensor TMP36, mientras que la implementación física se realizó con el sensor LM35. El proyecto fue desarrollado en Arduino IDE utilizando lenguaje C/C++ como parte de la asignatura de Diseño Digital.
